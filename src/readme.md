# Useless Button

## How do I launch the game?
1. Navigate to the folder: `\release\`.
2. Launch the application `Butt.exe`.

## Having trouble launching the game?
Try installing the prerequisites.
1. Navigate to the folder: `\release\Engine\Extras\Redist\en-us\`.
2. Launch the application `UE4PrereqSetup_x64.exe`.
3. Follow the set up guide.
4. When this is done, try and launch the game again.

## There's no quit button!
We have realised that there is no exit button, however you can use `ALT+F4` to quit the game.
Alternatively, use the keyboard shortcut `Ctrl+Shift+Escape` to open the task manager.
Alternative alternatively, use `Ctrl+Alt+Delete` to open the security options screen.

## What is the folder structure?
The `\assets\` folder is for the game's sounds and texture asset files. You can also find the application icon here.
The `\sauce\` folder is for the game's source project that you can open in Unreal Engine.
